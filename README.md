# Linux-ssh-Lab
Practicing secure remote access and authentication using SSH on Linux
This project documents my hands-on exprience with SSH (secure shell),including remote access ,key-based authentication ,and SSH client configurations.

Topics coverd
  OpenSSh installation
  SSH service management 
  SSH key generation
  Public and Private key
  Passwordless authentication
  SSH client configurations
Commands used
  sudo apt install openssh-server
  systemctl status ssh
  ssh-keygen -t ed25519
  ssh-copy-id user@test
  ssh user@test
What I learned
  How SSh enables secure remote access
  The diffrences between public and private keys
  Passwordless authentication using SSh keys
